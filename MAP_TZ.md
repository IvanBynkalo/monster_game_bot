```
assets/world_map.png
```

## Технические требования
- Формат: PNG или JPEG (сохранить как .png)
- Размер: **2048×1536 px** (горизонтальная ориентация 4:3)
- Стиль: фэнтези-карта в стиле старинных рукописных атласов

## Локации и их расположение

```
                    [🌌 Разлом эмоций]
                   /          \
          [🏔 Хребет бурь]   [🔥 Вулкан ярости]
               |                    |
          [🏜 Пустыня костей]  [⛰ Каменные холмы]
               |                /        \
          [🏛 Древние руины] [🌿 Изумрудные поля] [🕸 Болота теней]
               |                |                      |
          [🌲 Тёмный лес] ------+-------- [🌫 Болото теней]
               |
          [🏙 Сереброград]  ← стартовая точка, центр-юг
```

| Локация | Позиция | Визуал |
|---------|---------|--------|
| 🏙 Сереброград | Центр-юг | Каменный город с башнями, мост через реку |
| 🌲 Тёмный лес | Север от города | Густой тёмный лес, синеватый туман |
| 🌿 Изумрудные поля | Северо-восток | Зелёные луга, редкие деревья |
| ⛰ Каменные холмы | Восток | Серые скалы, жилы кристаллов |
| 🕸 Болота теней | Запад | Тёмная болотная вода, мёртвые деревья |
| 🌫 Болото теней | Северо-запад | Чёрная вода, фиолетовый туман |
| 🏛 Древние руины | Север | Полуразрушенные колонны, светящиеся символы |
| 🏜 Пустыня костей | Северо-восток | Выжженная пустыня, кости |
| 🔥 Вулкан ярости | Далёкий восток | Кратер вулкана, лава |
| 🏔 Хребет бурь | Далёкий север | Горный хребет, молнии |
| 🌌 Разлом эмоций | Крайний север | Мистическая трещина, разноцветное свечение |

## Легенда карты (нарисовать в углу)
- Пиктограмма города
- Пиктограмма опасной зоны (череп)
- Пиктограмма мистической зоны (глаз)
- Компас (стороны света)
- Масштаб (1 у.е. = условная единица пути)

## Стиль
- Старинная бумага с потёртостями по краям
- Рукописные названия локаций
- Иллюстрации существ в незаселённых зонах
- Горы нарисованы как в средневековых картах (≈ профиль)
- Леса — группы деревьев сверху
- Вода — синяя с волнами
