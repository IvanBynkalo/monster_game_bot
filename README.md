# Monster Emotions Bot — v3.2

Telegram RPG-бот где монстры рождаются и мутируют под влиянием эмоций игрока.

## Быстрый старт

```bash
pip install -r requirements.txt
export BOT_TOKEN="your_bot_token_here"
export ADMIN_IDS="123456789"
python bot.py
```

База данных создаётся автоматически в `data/game.db` при первом запуске.
Новые таблицы добавляются автоматически через ленивую миграцию.

### Основные (кнопки меню)
| Кнопка | Действие |
|--------|----------|
| 🌲 Исследовать | Вылазка: звери (62%), события (30%), монстры (8%) |
| 🧭 Переместиться | Переход в другую локацию |
| 🐲 Мои монстры | Список монстров с inline-управлением |
| 🎒 Инвентарь | Предметы и ресурсы |
| 👤 Персонаж | Профиль с 7 inline-вкладками |
| 📂 Ещё | Квесты, Сюжет, Лечение, Реликвии, Кодекс |

Появляется автоматически при входе/переходе в локацию:

| 🕳 Подземелье | Если есть подземелье |
| 🗺 Карта / Перемещение | Всегда |
| 🌌 Ритуал рождения | Только в Сереброграде и Разломе эмоций |

## Игровые механики

### Исследование регионов (Картограф)
- 0–100% на каждый регион, у каждого игрока свой прогресс
- За вылазку: 1–3% (зависит от уровня Картографа)
- На каждые 5% — событие: тайник, лор, бонус, подсказка
- На 100% — уникальный региональный бонус
- На 100% + раз в неделю — специальный квест

### Звери и монстры
- **62%** вылазок — встреча со зверем (5 видов на каждую локацию)
- **8–18%** — эмоциональный монстр (редко, ценно)
- **~30%** — событие (тайники, аномалии, погода)
- Звери дают ресурсы и лут, редкие — трофеи
- Монстров можно ловить и прокачивать

### Рождение монстров
- Авторождение **отключено** — только ручной ритуал
- Нужно накопить 30–50 единиц одной эмоции
- Места: **Ремесленный квартал** Сереброграда (max epic) или **Разлом эмоций** (max mythic)
- `/birth` — провести ритуал, `/birth_panel` — посмотреть накопление

### Бой
- Все кнопки боя — **inline** на сообщении (не в reply-меню)
- После боя reply-меню возвращается к меню локации
- Типы монстров влияют на урон (таблица слабостей)
- Комбо-мутации дают бонусы ATK/DEF и спецэффекты

### Мастер ловушек (Ремесленный квартал)
10 рецептов в трёх категориях:
- **Ловушки:** Простая, Ядовитая, Морозная (skip counter), Взрывная
- **Приманки:** Лесная (+15% зверей), Редкая (+25% редких)
- **Снаряжение:** Ошейник охотника, Теневой плащ, Огненный амулет

### Монетизация
- **Telegram Stars** — флаконы эмоций, энергия, слоты, сезонный пасс
- **P2P-рынок** — 8% комиссия с каждой сделки
- **Сезонный пасс** — 300⭐, двойные награды за задания
