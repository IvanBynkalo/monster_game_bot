# Monster Game Bot

Текущая версия: v2.5

Что улучшено:
- исправлено переключение активного монстра
- главное меню помещается на экран телефона без длинного скролла
- экран перемещения теперь показывает только переходы
- админ-панель больше не мешает обычным кнопкам

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```
